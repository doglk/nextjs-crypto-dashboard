## ðŸ“¦ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/reymundvirtus/nextjs-crypto-dashboard.git
cd nextjs-crypto-dashboard
```

### 2. Install dependencies

```bash
npm install
# or
yarn
```

### 3. Run the development server

```bash

# or

```
